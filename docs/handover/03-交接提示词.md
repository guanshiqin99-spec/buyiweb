## 项目关键事实速查（给接手者的背景，无需复述）

- 结构：`buyi-dictionary-vue/`（Vue3+Vite5+Pinia，hash 路由，vite-plugin-singlefile 单文件构建）＋ `BuyiDictionaryApp-main/BuyiDictionaryApp-main/`（微信小程序 + backend/ NestJS+TypeORM）＋若干归档目录。
- 本地启动：根目录 `启动竞赛包.bat`（唯一路径正确的 bat）；后端 `npm run start:dev`（sqlite），前端 `npm run dev`（vite 代理 /api → 127.0.0.1:3000）。
- 生产：前端 Cloudflare Pages（构建时无 VITE_API_BASE_URL，回退同源 `/api`，由 Pages Functions 代理转发）；后端待迁移到稳定 HTTPS（sealos-deploy.yml 备好）。
- AI：后端 miniapp-agent 模块（DeepSeek，SSE，`MiniappJwtGuard` 鉴权，`/ask` 限流 10 次/分钟）；前端 `src/utils/agentStream.js` 裸 fetch 读 SSE。
- 小程序走云函数 apiProxy 直连后端 IP（`cloudfunctions/apiProxy/index.js:9`），后端迁移后该地址要同步改。
