# 素材来源与展示边界

## 文化文字资料

- 布依族服饰、斗纹布、铜鼓十二调的页面说明链接到中国非物质文化遗产网对应公开条目。
- “布依族民歌（好花红调）”展项链接到中国非物质文化遗产网公开条目：<https://www.ihchina.cn/art/detail/id/12668.html>。
- 合成调值钢琴仅展示轮廓，不可标注为真实例词录音或田野录音。

## 图像

- 首页山景及现有氛围背景只承担情绪和空间导览，不作为文化事实、工艺资料或展品照片。
- 页面正文中的文化解释必须保留可点击出处；AI 图不可成为事实依据。

## 音频

- `public/audio` 的随包曲目在民歌页逐首显示演唱者、来源状态和演示授权说明。
- 在公开发布、直播或商业传播前，须由素材负责人补充原始采集来源、授权主体、授权范围与有效期；未完成核验的音频只可在当前竞赛演示包内使用。
