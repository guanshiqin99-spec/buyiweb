# 前端动效优化方向

## 当前问题

1. **首页内容区块没有进入动效**
   `Home.vue` 中各 section（language-exhibit、craft-exhibit、sound-exhibit、participation-exhibit、closing-exhibit）目前是直接进入，没有 reveal 或 stagger，导致长页面滚动时显得呆板，也缺少前后节奏。

2. **动效种类过多，互相打架**
   首页同时存在 prelude 开场、背景粒子、液玻高光、视差滚动，注意力被反复拉扯；但真正需要的 reveal 反而缺失。

3. **粒子与光晕装饰化**
   背景粒子数量多、旋转幅度大，漂浮感强；光标光晕覆盖范围过大，在工具页和高密度列表上变成视觉噪音。

4. **工具页被 PageShell 全包，背景与结构不合适**
   Favorites、Learn、Record、Settings、Profile 等工具页均使用 `PageShell`，默认带全出血背景图、遮罩、粒子与视差。例如 `Settings.vue` 使用 `bg-profile.png` 作为背景且 `particle-density="8"`，这与「设置」页面的工具属性冲突，造成背景喧宾夺主、UI 结构被压在大面积氛围图之下。

5. **动效节奏不统一**
   有的动画 200ms，有的 800ms，有的无限循环；没有共享的时长与缓动系统，页面显得碎。

6. **缺少 reduced-motion 兜底**
   部分粒子、光晕、呼吸动画未在系统「减少动效」设置下静默。

## 优化方向

### 方向一：只保留 3 种有目的的动效

- **Hero 入场**：首页首屏文字/CTA 一次性 stagger 淡入，总时长 ≤ 700ms。
- **滚动 reveal**：内容区块进入视口时轻量淡入上滑，只触发一次。
- **操作反馈**：按钮 hover/active 用 scale/brightness/shadow 微变化，≤ 200ms。

其余装饰性动效（全站光标光晕、强液玻镜面扫光、反复漂浮粒子）降级或移除。

### 方向二：工具页退出 PageShell 全包模式

`PageShell` 适合「大标题 + 氛围背景 + 内容叠在背景上」的叙事页。工具页应改为：

- 顶部保留简洁页面标题与面包屑/返回，不再使用全出血 hero 背景图。
- 内容区使用窄居中布局（如 `max-width: 680-820px`），背景用纯色或极淡纹理。
- 完全移除粒子和视差；需要品牌氛围时，仅在顶部小面积使用品牌色渐变条或装饰纹样。

例如 Settings/Profile/Favorites/Record/Learn 应改为：

```
[AppHeader]
[页面标题区：浅色背景 + 小标题 + 一句话说明]
[内容区：设置分组 / 统计卡 / 列表，居中，卡片用 quiet 材质]
[AppFooter]
```

### 方向三：区分页面性格

| 页面类型 | 代表页面 | 动效处理 |
|---|---|---|
| 叙事页 | Home、Songs、Culture | 保留低强度视差（系数 0.08-0.12）、少量粒子（≤ 6） |
| 工具页 | Dictionary、Settings、Profile、Favorites、Record、Learn | 退出 PageShell 全包，关闭粒子与视差，背景静态，优先可读 |

### 方向四：统一节奏系统

在 `variables.css` 建立三档时长：

- `--duration-fast: 150ms` —— 微交互、hover、active
- `--duration-base: 250ms` —— reveal、状态切换
- `--duration-slow: 500ms` —— Hero 入场、重大状态变化

统一缓动：`cubic-bezier(0.22, 0.61, 0.36, 1)`。

## 关键检查点

- [ ] 首页 Hero 文字/CTA 有 stagger 入场，总时长 ≤ 700ms
- [ ] 首页 language-exhibit、craft-exhibit、sound-exhibit、participation-exhibit、closing-exhibit 滚动进入时有 reveal 动画
- [ ] 滚动 reveal 只触发一次，时长 250-400ms
- [ ] 按钮 hover/active 使用 brightness/scale/shadow 之一，过渡 ≤ 200ms
- [ ] Settings、Profile、Favorites、Record、Learn 退出 `PageShell` 全包模式，改为简洁标题 + 居中内容布局
- [ ] 工具页关闭粒子与视差，背景静态
- [ ] 叙事页粒子 ≤ 6、视差系数 0.08-0.12
- [ ] 光标光晕仅桌面核心操作按钮生效，触屏与工具页隐藏
- [ ] `prefers-reduced-motion: reduce` 下所有 reveal、粒子、光晕、视差停止或隐藏
- [ ] 构建无错误无新警告
